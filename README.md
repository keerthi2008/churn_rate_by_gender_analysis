# churn_rate_by_gender_analysis

#📌 Project Overview

This project analyzes customer churn data to identify patterns, trends, and key factors that influence customer retention. The analysis is performed using Microsoft Excel with Pivot Tables, Charts, and  techniques.

#🎯 Objectives

Understand customer churn behavior

Identify key factors affecting churn

Visualize data using charts

Provide actionable insights and recommendations

#📁 Dataset

Dataset Name: Telco Customer Churn Dataset

Features include:

Gender

Senior Citizen

Tenure

Monthly Charges

Contract Type

Services (Internet, Tech Support, etc.)

Churn (Yes/No)

#🛠️ Tools Used


Microsoft Excel

Pivot Tables

Charts (Bar Chart, Column Chart, Histogram, Scatter Plot)

#📊 Tasks Performed

✅ Task 1: Data Cleaning

Removed blank values

Checked for duplicates

Converted categorical values if needed

✅ Task 2: Churn Count Visualization

Created column chart to compare churn vs non-churn

Added data labels and formatting

✅ Task 3: Monthly Charges Distribution

Created histogram for Monthly Charges

Used appropriate bin sizes

Analyzed distribution pattern

✅ Task 4: Churn Rate by Gender

Created Pivot Table:

Rows → Gender

Columns → Churn

Values → Count of Churn

Built Clustered Bar Chart

Applied different colors for churn categories


#📈 Key Insights

Customers with month-to-month contracts have higher churn

Higher monthly charges lead to increased churn

Lack of tech support and security services increases churn

Certain gender segments show slightly higher churn patterns

#💡 Recommendations

Offer discounts for long-term contracts

Improve customer support services

Provide bundled service packages

Focus on high-risk customers with high monthly charges
